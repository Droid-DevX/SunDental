##  About The Project

Welcome to the official frontend application for **Sun Dental**, located in the heart of Kalaburagi, Karnataka. 

This highly interactive web application is crafted with surgical precision to provide a **premium, dynamic, and immersive** digital experience. From the high-quality 3D tooth models to the elegant glassmorphism digital storefront, everything speaks modern clinical excellence.

###  Premium Highlights

- **Aesthetic Clinical Design:** A harmonious blend of Navy, Sky Crystal, and Mint colors. Combined with premium modern typography (`Syne`, `DM Sans`) reflecting pure elegance and futuristic vibes.
- **Fluid & Cinematic Animations:** Powered by **Framer Motion**, delivering ultra-smooth scroll reveals, custom glow cursor effects, spring physics, and seamless transitions.
- **Breathtaking 3D Canvas:** Interactive WebGL particle backgrounds, custom 3D floating tooth models, and orbiting dental tools engineered with **Three.js** and **React Three Fiber**.
- **Performance First:** Built with bleeding-edge tools—**React 18** and **Vite**, boasting near-instant loading times and blazing fast Hot Module Replacement (HMR).

##  Getting Started

Follow these instructions to set up your local development environment.

### Prerequisites

Ensure you have the latest stable version of [Node.js](https://nodejs.org/) installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Navigate into the directory:**
   ```bash
   cd SunDental
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the local development server:**
   ```bash
   npm run dev
   ```

5. **Feast your eyes:**
   Open your browser and navigate to `http://localhost:5173`.

##  Architecture & Structure

```text
src/
├── components/       # Reusable, atomic React components (Navbar, Cursor, LoadingScreen)
├── sections/         # Main page sections (Hero, About, Services, Location, Contact)
├── three/            # Three.js 3D environment & WebGL configurations (ToothModel, HeroScene)
├── index.css         # Global styles and tailwind directives
├── App.tsx           # Application orchestrator mapping routing & context
└── main.tsx          # React DOM mounting and global providers
```
